# Negative Harmony

ブラウザ単一 HTML のネガティブハーモニー変換 toy。キーと度数進行を入れると、各コードをトニック–ドミナント軸（トニック+3.5半音）で鏡映し、「裏の進行」を生成して before/after を鳴らし比べる。長調が短調の影に、ドミナントが別の引力に化ける感覚を耳で掴む。

理論 toy 連作の一員。reharm-roulette の親戚だが別理論（鏡映変換）。Tone.js のみ依存。アーティスト名非依存。

鏡映式: tonic を T とすると `r(p) = (2T + 7 − p) mod 12`（軸 = T と T+7 の中点 = T+3.5）。
