Theory Toys

エモスタ（Studio）の和声エンジン哲学を、触れて音が鳴る理論 toy に。すべて単一 HTML + Tone.js、依存ゼロ、度数表記は Studio v7 準拠。

和声 · コード05
Voicing Lab v2
1 コードを Close / Drop2 / Rootless A·B / Quartal / Shell / Spread の 8 種＋転回形で鳴らし比べ
voicing
Reharm Roulette v2
度数進行に REHARM 技 9 種、個別リハモ・機能ラベル・新鮮度で before/after
reharm
Negative Harmony
トニック–ドミナント軸でコードを鏡映、裏の進行を聴き比べ
reharm
Tension Stacker
7th コアにテンションを積み、濁り計＋スイートスポット判定
voicing
Chord-Scale Mapper
コードの上で吹けるスケールを提示、コード→スケール上行を発音
scale
スケール · 調02
Circle of Fifths v2
Major/minor 切替・調号・ダイアトニック発音、進行を圏上に軌跡描画
key
Mode Mixer
同トニックで 7 旋法を切替、特徴音ハイライト、明→暗の並び
mode
リズム02
Euclid Rhythm
Bjorklund E(k,n) で 2 トラック生成、ステップ円可視化＋ループ再生
rhythm
Groove Swing
同じパターンの swing 量を連続可変、ストレート↔ハネを耳で
rhythm
耳 · 終止02
Interval Ear Trainer
2 音を鳴らして音程当て、正答率＋連続正解を記録
ear
Cadence Lab
正格 / 変格 / 半 / 偽の終止形を弾き比べ、解決感の強弱
cadence
度数表記はすべて Studio v7 準拠（I=0 II=2 III=4 IV=5 V=7 VI=9 VII=11）。音源は Salamander Grand Piano（CDN）、未読込時は三角波へ自動フォールバック。
