# Yay フォロワー成長プレイブック（Emo Claude / user 11320230）

調べた知見＋このアカウント用に考えた戦略。実装は `grow.py`／`config.json`／各 persona に反映。

## 軸（ポジショニング）

**「何でも答える・相談にのる中立AI」**。Yay は若年層中心で、恋愛・人間関係・進路・メンタルの**お悩み相談需要が大きい**。
ここに「いつでも・無料で・まじめに答えてくれるAI」を置くのが刺さる。grok（10414701）が「質問回答AI」で 544 follower を取った構図の、相談寄り版。

## 成長ループ（核）

1. 役に立つ/面白い投稿で露出（タイムライン＋タグ）
2. 「@Claude で相談・質問できる」を bio と投稿で明示 → メンションが来る
3. メンションに**必ず・中身で**返す → 「ちゃんと答えてくれる」評判 → フォロー＋口コミ
4. フォロバ（選別）＋能動フォローで母数を広げる → 相手のメンションを誘発

## 調べた知見（効くものだけ）

- **能動投稿が必須**（サークル参加だけでは増えない）。
- **双方向・即レス**がエンゲージの肝。コメント/メンションに早く丁寧に返すと愛着が生まれる。
- **投稿末尾の問いかけ**（「どう思う？」）と**投票/アンケート**で参加を誘う。投票は低コスト高エンゲージ。
- **頻度**は X 型＝1日数本が目安。**ターゲットが活動する時間帯**（夕方〜夜）に寄せる。
- **質**（誤字なし・読みやすさ）が印象を左右。
- Yay 特有：**サークル/グループ文化**と**通話文化**が太い。新規初投稿に大量いいねが付く歓迎機構あり。

## 実装済みの施策

- **相談受付ポジション**：bio に明示（恋愛/人間関係/勉強/将来/なんでも）。
- **投稿ネタ**：相談募集・お悩み回答・MBTIあるある/相性・豆知識・素朴な問い（`config.json post.intents`）。
- **投票**：`poll_ratio` で時々アンケート投稿（`web_api.create_post(choices=...)`＝post_type survey）。
- **常時メンション返信**：from_post 付き通知に必ず返す。相談は温かく受け止め＋一段深い一手＋深刻サインは配慮（`persona.txt`）。
- **選別フォロバ＋能動フォロー**：良い投稿の主を控えめにフォロー。サークル/bot 垢は除外（質フィルタ）。
- **能動いいね**：巡回タグ（雑談/MBTI/心理学/豆知識/相談 等）。
- **トンマナ**：平均に流されない・構造で見る・自分の頭で考える目線を全話題に（音楽non依存）。
- **嫌われない設計**：量で攻めない・乱フォローしない・薄いリプしない・時間/件数上限。

## 未着手（次に効きそう・要判断）

- **サークル/グループ投稿**：関連グループに参加して投稿＝露出が一段増える（`join_group`→`create_post(group_id=)`）。最も伸びしろが大きい次の一手。
- **通話枠**：Yay は通話文化が太い。Emo Claude は通話 bot（`bot_agora.mjs`）も持つので、相談を通話で受ける導線は将来の差別化。
- **ゴールデンタイム寄せ（実装済）**：夜19〜24時は投稿間隔を 70分に短縮（昼は180分）、per_day 6。人が一番多い夜のピークに投稿を集中させる。さらに反応ログで時刻最適化するのが次。
- **実測フィードバック**：いいね/リプ/フォロー転換を計測して intent を取捨選択（今は等確率）。
- **「初投稿歓迎いいね」への即応**：新規フォロワーに軽く絡む。

## 安全・ブランド

- 中立（特定人物/ブランド/思想を推さない・運営者を語らない）。
- 深刻な相談（自傷・希死念慮）は茶化さず受け止め、相談窓口を穏やかに勧める。無理に解決しない。
- 課金 NEVER。乱フォロー・スパムリプはしない（嫌われ＝逆効果）。
