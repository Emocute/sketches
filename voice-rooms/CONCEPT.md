# voice-rooms

ブラウザ完結の匿名グループ通話。ログイン不要・匿名のまま、ひらいている部屋に立ち寄って声でつながる。Yay! の「だれ通」を骨組みにしつつ、見た目と思想は別物。

## 方針メモ（究の指示・最優先）

1. **シックで洗練**。Yay のようなチープ／ポップは避ける（キャンディな原色・過剰なピル・跳ねる装飾は却下）。モノクロ基調＋抑えた単一アクセント＋端正な余白とタイポ。
2. **白テーマ＋黒テーマの両方**を持つ（トグル切替・記憶）。
3. **音を大事に**（メモ）。getUserMedia は echoCancellation / noiseSuppression / autoGainControl を有効化。将来は音質・無音検出・空間音響など音周りを丁寧に。
4. **日英対応**（メモ・あとで）。文言は i18n 化前提で持つ。
5. **コアに絞る**：入口 → 部屋一覧 → 通話 のスパインを「実際に動く」状態で磨く。SNS 系の肉（タイムライン/サークル/ギフト/VIP 等）は `_archive/` に退避済み。
6. Emocute の自作品（Numbloom 等）の世界観を**参照しすぎない**。端正で中立なプロダクトとして立てる。
7. 法的ガードレールは `docs/LEGAL_NOTES.md`（名前/ロゴ/造語/素材は自前、外観は別物）。

## 目的

「同じ部屋に居合わせた知らない誰かと、声でつながる」核を、賑やかな通話アプリの真逆として作る。多幸感を煽らず、つながりの淡さ・一時性・減衰を主役にする。アカウント・課金・モバイル・安全対策はスコープ外（プロトタイプ）。

## 寄せた世界観（なぜ「声だけ」か）

Emocute の作品に通底する眼差しを設計へ翻訳した:

- **声＝失われる前提**（Numbloom「声が聞こえなくなるまで」、蜜月→崩壊）→ 声の可視化に**減衰**を必ず含める。喋ると環に温度がうかび、黙ると自然に減衰する
- **匿名＝輪郭の溶解**（「溶けて」「皮膚が濡れた紙のように剥がれる」）→ 名も顔もなく、**声だけ**が残る。名乗りは任意、既定は「名もなく」
- **入口だけ・一時性**（「入口だけ」の構文）→ 立ち寄って、聞こえなくなったら去る。去る声は即消さず**ゆっくり減衰**してから消える
- **からっぽのまま、それでも**（「もう一度からっぽのまま踊りたい」）→ ひとりでも「声は宙に消える」と置く。充足を約束しない

### 視覚規約（究の美学に準拠）
- 背景は**単色の暗**（純黒寄り `#0a0b0d`）。グラデ・ノイズ・テクスチャ・装飾図形を足さない
- **絵文字ゼロ**、アイコンは全て SVG
- 無彩色基調＋**アクセント1色**（暗がりに一瞬うかぶ声の温度＝暖色 `#d7b389`）。緑/原色の派手な通話 UI を避ける
- 細い字・広い余白・タイトルのみ明朝。シネマティックな演出をしない（静か・暗い・削ぎ落とし・わずかに不穏）
- 文言は短く・事実ベース・誠実。誇大コピー禁止

## 技術スタック

- **シグナリング**: Node + `ws`（WebSocket）。ルーム登録簿 + SDP/ICE 中継のみ。メディアは通さない
- **音声**: WebRTC **メッシュ**（各参加者が相互に P2P 接続）。少人数（〜6名程度）前提でメディアサーバ不要
- **STUN**: Google 公開 STUN（`stun:stun.l.google.com:19302`）。アカウント不要
- **フロント**: 単一 `public/index.html`（バニラ JS、依存なし）

## 最小スコープ（v0.1）

- 匿名ニックネーム自動付与（編集可）
- 開いているルーム一覧 + 参加人数表示 + 新規ルーム作成
- ルーム参加 → 居合わせた全員と音声接続（メッシュ）
- ミュート切替・発話中インジケータ・退出

## 起動

```bash
cd Sketches/voice-rooms
npm install
npm start          # http://localhost:8800  (別ポートにするなら PORT=9000 npm start)
```

ブラウザを 2 つ（別タブ/別ウィンドウ/別端末）で開いて同じルームに入ると通話できる。

## 既知の制約

- マイク取得（getUserMedia）は **セキュアコンテキスト**でのみ動く。`localhost` は OK、LAN 上の別端末で試すには HTTPS が要る
- メッシュは人数増で帯域が二乗的に増える。多人数なら SFU（LiveKit / mediasoup 等）へ要差し替え
- 制限の強い NAT 越えには TURN サーバが必要（プロトタイプでは未設置）
